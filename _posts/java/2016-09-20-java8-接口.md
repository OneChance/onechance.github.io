# 默认方法

```
default void remove() {
    
}
```
#### 默认方法可以扩展接口而不会造成实现类报错，实现类可以访问这个方法

# 在接口中定义静态方法

```
public static <T, U extends Comparable<? super U>>
    Comparator<T> comparing(Function<T, U> keyExtractor) {
    return (c1, c2) -> keyExtractor.apply(c1).compareTo(keyExtractor.apply(c2));
}
```
